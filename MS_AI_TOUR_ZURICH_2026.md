## Persona-Based Guidance

### Director of IT & IT Project Managers
**Focus:** Strategy, ROI, and Operationalization.
- **Key Takeaway:** Move from experimentation to execution using the **Agent Factory** model.
- **ROI Measurement:** Use the new **Engagement Backend** and **ROI Reports**. Measure success by "Deals Created," "Time Saved in CRM Filling," and "Automated Document Matching."
- **Application:** Implement a "Build AI with AI" approach. Use Agentic DevOps to assign tasks directly from GitHub to Agents. Focus on "Ownerless Agents" for persistent background processes.

### Cyber Security Director & Data Owners
**Focus:** Security, Compliance, and Data Integrity.
- **Key Takeaway:** AI Security is built on **Zero Trust** and inherits existing Microsoft 365 permissions and **Purview** sensitivity labels.
- **Security Tools:**
  - **Entra Policies:** Detect and block risky agents based on behavioral criteria.
  - **Purview AI Dashboard:** View why specific messages or data access attempts were blocked.
  - **Agentic Defense:** Deploy "Autonomous Blue Teams" to counter "Ambient Red Teams."
- **Application:** Ensure all data sources (Workday, Oracle, etc.) are connected via secure connectors that respect user-level permissions.

### AI Architect, Cloud & Infrastructure
**Focus:** Architecture, Integration, and Performance.
- **Key Takeaway:** The **Microsoft IQ Stack** provides a unified intelligence layer across the enterprise.
- **Architectural Pillars:**
  - **Foundry IQ:** The knowledge base for RAG (Retrieval-Augmented Generation) using "Agentic Retrieval."
  - **Fabric IQ:** The real-time state of the business (OneLake, Power BI, SQL).
  - **Work IQ:** The context of collaboration (Teams, Outlook, SharePoint).
- **Application:** Utilize the **Model Context Protocol (MCP)** to connect Dynamics 365 and external systems to agents. Explore **Majorana 1** for future quantum-accelerated workloads.

## The Intelligence Stack (The IQ Layers)

The "IQ Stack" represents the intelligence outcomes Microsoft wants organizations to achieve. These are not separate SKUs but architectural layers.

| Layer | Purpose | Data Sources |
| :--- | :--- | :--- |
| **Work IQ** | Understands *how* people work and collaborate. | Teams, Outlook, M365 Documents, Graph API. |
| **Fabric IQ** | Represents the live *state* and actions of the business. | OneLake, Fabric, Power BI, SQL, Real-time telemetry. |
| **Foundry IQ** | Provides curated *institutional knowledge*. | SharePoint, Web, Blob Storage, MCP Servers. |

## The Agentic Ecosystem

### Agent Factory
The "supply line" for AI. A unified operating and commercial model to scale AI agents across the enterprise. It matches the right platform (Copilot Studio for low-code, Microsoft Foundry for pro-code) to the right builder.

### Agent 365
The **Control Plane** for governance. It allows AI Admins to:
- Control who can create and deploy agents.
- Monitor costs per group/tenant.
- Manage "Ownerless Agents" and background workflows.
- Manage agents from both Microsoft and third-party providers.

### Copilot Studio & MCP
- **MCP (Model Context Protocol):** Standardizes how agents interact with external data (e.g., Workday, Oracle).
- **Cowork:** Allows multiple agents to run in parallel, with a "Facilitator" agent acting as a third member of the team.

## Security, Governance & ROI

- **Secure by Design:** AI and security are now two sides of the same platform.
- **Confidentiality Inheritance:** Sensitivity labels from Purview are automatically respected by Copilot and Agents.
- **Sovereign Cloud:** Options for **Sovereign, Private, and National clouds** are available for highly regulated sectors (e.g., FINMA).
- **ROI Reports:** Automated tracking of efficiency gains. For example, quantifying how many documents were matched by AI vs. manual accounting.

## Industry Spotlight: Financial Services
*Session: BRK202-CH*

- **LGT (Head of Business & Innovation):** 5-year transformation focusing on recording customer conversations to automatically fill CRM data.
- **Migros Bank (Head of Data Automation):** Focusing on OCR AI recognition for incoming documents. The goal is to shift accountants from "matching documents" to "curating data."
- **Vaudois (CFO):** Emphasized that guardrails and employee training are critical in a FINMA-regulated environment.
- **Risk Management:** "No agentic AI without knowing the topic." Human-in-the-loop remains essential for high-risk financial decisions.
